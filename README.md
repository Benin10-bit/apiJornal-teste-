# Jornal Teresa Teste

Este repositório é voltado para guardar o código base de teste da API e do frontend do Jornal Teresa.
Serão alteradas algumas coisas do código, inicialmente, trocaremos o fastify pelo express e utilizaremos uma ORM chamada Prisma. Incrementaremos criação de usuário para todos (não apenas ADMINS) e utilizaremos para o novo front end o Next.JS, com typescript e tailwindcss.
